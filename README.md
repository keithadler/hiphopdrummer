# 🥁 Hip Hop Drummer

**Generate unique hip hop beats from scratch — 37 styles, 9 instruments, free, runs in your browser.**

### **[▶ Try It Now — hiphopdrummer.com](https://hiphopdrummer.com)**

![Version](https://img.shields.io/badge/version-1.64-blue) ![License](https://img.shields.io/badge/license-MIT-green) ![No Account](https://img.shields.io/badge/account-none_required-brightgreen) ![Offline](https://img.shields.io/badge/works-offline-orange)

Every beat is assembled in real time from hundreds of musical rules — not selected from presets, not generated by AI. Drums, bass, electric piano, synth pad, lead, organ, horns, vibes, and clavinet all play together, following the same chord progressions and reacting to each other. Full song arrangements with intros, verses, choruses, breakdowns, and outros. Export WAV, MIDI, MPC patterns, chord sheets, and PDF beat sheets. No account, no cloud, no subscription.

**The beat is the harmonic foundation — you bring the melody.** Your rap, your sample, your guitar, your scratch. That's how hip hop has always worked.

## Who It's For

- **Rappers** — Hit play and freestyle. Export the WAV for demos. The flow guide tells you where to land your syllables.
- **Producers** — Export MIDI, swap the sounds, customize in your DAW. The patterns are musically correct — you're not starting from scratch.
- **DJs** — Full song arrangements with breakdowns for scratching. Export WAV, load on a deck.
- **Musicians** — Chord sheets, bass MIDI, EP voicings. Play along or study the parts.
- **Students** — Click any cell to understand its velocity. Read the analysis. Every beat is a free lesson.

## 40 Years of Production Knowledge

312 curated kick patterns. Named player profiles (Premier, Questlove, Dilla, Lil Jon). Per-instrument swing. Modal harmony (Dorian IV for G-Funk, Phrygian bII for Memphis). Bass call-and-response. Tempo-aware generation. Dynamic arrangement arcs. Section-ending fills matched to each era. This is what 40 years of hip hop sounds like when you encode it into code.

## Quick Start

No install needed. Just open **[hiphopdrummer.com](https://hiphopdrummer.com)** in any browser.

```bash
# Or serve locally:
python3 -m http.server 8080
```

### Developer Setup

```bash
npm install          # install SpessaSynth + esbuild
npm run build        # bundle synth-bridge.mjs → synth.js
npm test             # run 18,000+ assertions
```

## Highlights

- **37 styles + 6 regional variants** — from old school 808s to modern boom bap revival
- **9 instruments** playing together with style-matched sounds (TR-808 for G-Funk, Brush Kit for Nujabes, etc.)
- **SpessaSynth audio engine** — high-quality GM SoundFont playback
- **Per-instrument swing** — hats, kick, ghost snares, bass each swing differently per style
- **Beat drops** — dramatic silences where ALL instruments drop out
- **Strict vs Improvise** — same part every time, or slight variations like a live band
- **Tap tempo** — double-click BPM or press T
- **Keyboard shortcuts** — Space, R, T, E, L, ←/→
- **PWA** — installable, works offline
- **Beat History** — 100 slots with favorites

## Testing

## Project Structure

```
├── index.html          App shell and layout
├── styles.css          Dark/light theme, responsive
├── patterns.js         Style data, kick libraries, swing tables
├── ai.js               Generation pipeline, arrangement builder
├── writers.js           Bar writers, fills, intro/outro
├── groove.js           Accent curves, humanization
├── bass.js             Bass generator + MIDI/MPC export
├── ep.js               Electric piano (30 features, 12 styles)
├── pad.js              Synth pad (10 features, 7 styles)
├── lead.js             Synth lead (G-Funk whistle)
├── organ.js            Organ (jazz/Nujabes)
├── horns.js            Horn stabs (boom bap/big/driving)
├── vibes.js            Vibraphone (Nujabes/jazzy)
├── clav.js             Clavinet (bounce/G-Funk)
├── analysis.js         Educational text generator
├── ui.js               Grid, arrangement editor, chord sheet
├── midi-export.js      MIDI/ZIP export, combined 9-instrument MIDI
├── daw-help.js         DAW setup guides (9 DAWs + MPC)
├── pdf-export.js       PDF beat sheet + chord sheet
├── beat-history.js     History storage and dialog
├── app.js              Main controller, dialogs, playback
├── synth-bridge.mjs    SpessaSynth integration (ES module)
├── synth.js            Bundled synth engine
├── tests.js            18,000+ automated assertions
├── sw.js               Service worker (offline PWA)
├── manifest.json       PWA manifest
├── robots.txt          Search engine directives
├── sitemap.xml         Search engine sitemap
└── GeneralUserGS.sf3   GM SoundFont (10MB)
```
